🎉 新增功能：
- 快速搜索窗口支持根据可见内容自动展开与收起：默认仅显示输入框，出现检索结果后自动展开列表。

🐛 问题修复：
- 修复快速搜索默认打开时，输入框下方透明空白区域可能阻挡桌面或其他应用鼠标操作的问题。
- 修复鼠标点击快速搜索结果后，左侧激活样式未及时跟随选中项更新的问题。
- 修复双击应用、书签或搜索结果时会直接执行打开操作的问题。现在单击或双击仅切换激活项，按 `Enter` 才会执行跳转或打开。
- 修复 Git 自动同步可能并发执行多个 Git 操作，导致仓库残留 `index.lock`、后续同步失败的问题。
- 修复 Git 自动同步任务已被新任务替代后，旧任务仍可能继续执行并产生状态误判的问题。
- 修复代码块使用 `scss` 或 `mermaid` 时，语法高亮可能重复输出 Shiki 语言未加载警告的问题。

🔧 优化改进：
- 优化快速搜索窗口交互稳定性，减少透明区域边界误判、鼠标操作异常和不必要的状态切换。
- 优化深度搜索按钮提示方式，降低透明窗口中的浮层干扰。
- 重构全局提示条、通知和确认弹框样式，统一适配浅色与深色主题，减少组件库默认样式带来的视觉割裂感。
- 统一反向链接更新、片段类型选择、数据迁移、Git 配置、Git 冲突处理和录屏导出等弹框的标题栏、遮罩、按钮与间距。
- 优化 Git 手动合并工作台的弹层背景、边框和阴影，使其与当前主题保持一致。
- 优化 Git 自动同步调度机制：串行执行仓库操作，忽略过期任务，并增强锁文件异常场景下的恢复能力。
