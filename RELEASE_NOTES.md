🎉 新增功能：
- 新增桌面常用文件搜索，可在快速搜索中检索桌面上的文档、图片、压缩包等文件。
- 新增搜索结果内联预览，支持代码片段、笔记、桌面文本文件、图片及部分 Office/PDF 文档的快速预览。
- 新增搜索结果分类标签与虚拟列表展示，支持按应用、书签、文件、代码片段、笔记等类型快速筛选。
- 新增桌面文件缓存与桌面目录监听，文件变更后可自动刷新索引。
- 新增文件操作能力，可从搜索结果中打开文件、定位到文件所在文件夹或选择其他打开方式。
- 新增应用“以管理员身份运行”入口。
- 新增托盘菜单“检查更新”入口，手动检查到新版本时可直接打开更新窗口。
- 新增截图撤销/重做快捷键支持，支持 Ctrl/Cmd+Z、Ctrl+Y、Shift+Cmd/Ctrl+Z。

🐛 问题修复：
- 修复手动检查更新只检查状态、不自动打开更新窗口的问题。
- 修复搜索结果历史排序在多来源结果 ID 冲突时可能不准确的问题。
- 修复搜索窗口关闭时旧预览窗口残留、焦点和状态清理不一致的问题。
- 修复部分窗口创建、剪贴板、截图贴图等流程中异常处理不完善导致的潜在崩溃问题。
- 修复 Markdown 预览中本地图片、绝对路径、file 链接等资源无法正确显示的问题。
- 修复截图窗口关闭后缓存资源未完全清理，可能影响下次截图的问题。
- 修复扫描进度未统计桌面文件数量的问题。

🔧 优化改进：
- 重构快速搜索结果区域，提升大量结果下的滚动性能和键盘操作体验。
- 优化搜索窗口布局、结果高亮、类型标识、快捷键提示和无结果提示。
- 优化应用启动时资源扫描流程，将应用、书签、桌面文件缓存统一初始化。
- 优化日志输出，减少高频调试日志和文件监听日志噪音。
- 优化窗口管理与加载过渡逻辑，提升配置窗口打开稳定性。
- 优化截图界面视觉效果，包括遮罩透明度、提示文字、尺寸提示和文本输入框样式。
- 优化编辑器与内容列表相关界面，减少无用依赖和提升交互一致性。
