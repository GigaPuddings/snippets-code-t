# v1.4.2 更新日志

🎉 新增功能：
- **Git 同步重构**：使用系统 Git 命令实现同步，支持任意 Git 仓库（GitHub、GitLab、Gitee 等），支持分支管理、冲突检测和手动合并
- **多工作区支持**：可同时管理多个笔记库，切换工作区更便捷
- **优化的搜索索引**：基于内存索引的全文搜索，支持百万级文件快速搜索
- **编辑器增强**：
  - 本地图片扩展：支持插入本地图片
  - 自定义回车行为：Enter 键智能换行
- **截图功能增强**：支持导入本地图片进行标注
